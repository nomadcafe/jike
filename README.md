# 即刻News

![](/public/og-image.png)

**优雅地阅读实时热门新闻**

🌐 **在线访问**：[https://jike.news](https://jike.news)

即刻News 是一个专业的实时新闻聚合阅读平台，致力于为用户提供最新、最全面的新闻资讯。平台聚合了来自全球各地的优质新闻源，涵盖科技、财经、国际、娱乐等多个领域，让用户在一个界面中轻松浏览所有重要新闻，告别频繁切换多个网站的烦恼。

## 关于即刻News

即刻News 是一个现代化的新闻聚合阅读器，通过智能抓取和实时更新技术，将分散在各个平台的新闻内容集中展示。无论是科技动态、财经资讯、国际新闻还是娱乐热点，都能在这里第一时间获取。平台采用优雅简洁的设计，专注于提升阅读体验，让新闻阅读变得更加高效和愉悦。

## 功能特性

### 核心功能

- 🎨 **优雅的阅读界面** - 采用现代化设计理念，简洁美观的 UI 界面，专注于提升阅读体验，支持深色模式
- ⚡ **实时新闻更新** - 实时获取最新热点新闻，支持多种新闻源自动更新，确保信息时效性
- 🔍 **智能搜索功能** - 强大的全文搜索能力，快速定位感兴趣的新闻内容
- 📱 **PWA 渐进式应用** - 支持安装到设备，提供原生应用般的体验，支持离线访问

### 高级功能

- 🔐 **GitHub 登录同步** - 支持 GitHub OAuth 登录，实现跨设备数据同步，个性化阅读体验
- 💾 **智能缓存机制** - 默认缓存时长为 30 分钟，登录用户可强制刷新获取最新数据
- 🚀 **自适应抓取策略** - 根据内容源更新频率智能调整抓取间隔（最快每 2 分钟），优化资源使用，避免 IP 封禁
- 📊 **数据统计分析** - 集成 Google Analytics，提供详细的访问数据统计和分析

## 新闻源覆盖

即刻News 聚合了超过 50+ 个优质新闻源，涵盖多个领域和地区，确保用户能够获取全面、及时的新闻资讯。

- **国内新闻**：微博热搜、知乎热榜、36氪、腾讯新闻、百度热搜、今日头条、澎湃新闻、虎扑、贴吧等

- **财经资讯**：华尔街见闻、雪球、金十数据、MKTNews、财联社、格隆汇等

所有新闻源均经过精心筛选，确保内容质量和更新频率，为用户提供最优质的新闻阅读体验。

## 技术栈

- **前端**：React 19 + TypeScript + TanStack Router + TanStack Query
- **样式**：UnoCSS
- **后端**：Nitro (基于 H3)
- **构建工具**：Vite
- **数据库**：支持 Cloudflare D1、SQLite 等多种数据库
- **部署**：Cloudflare Pages / Vercel / Docker

## 快速开始

### 环境要求

- Node.js >= 20
- pnpm

### 安装依赖

```bash
corepack enable
pnpm i
```

### 开发

```bash
pnpm dev
```

### 构建

```bash
pnpm build
```

## MCP Server 支持

项目支持 MCP (Model Context Protocol) Server，可以在 AI 应用中集成。使用 jike.news 的配置示例：

```json
{
  "mcpServers": {
    "newsnow": {
      "command": "npx",
      "args": [
        "-y",
        "newsnow-mcp-server"
      ],
      "env": {
        "BASE_URL": "https://jike.news"
      }
    }
  }
}
```

你也可以将 `BASE_URL` 修改为你自己的域名。
